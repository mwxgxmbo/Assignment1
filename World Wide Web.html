World Wide Web

The WWW is an information system of interlinked hypertext documents and resources, accessible via the Internet using browsers. Invented by Tim Berners-Lee in 1989, it layers on top of the Internet — which is the physical infrastructure — providing a way to publish, link, and navigate information globally using HTTP, HTML, and URLs.
